# Prince-of-Persia
How to set up  AppleWin emulator on Mac

# Open the terminal and enter the following commands: 
 ``` brew install xquartz --cask ``` 
 ``` brew install wine-stable --cask ``` 

# Download AppleWin emulator:
[(https://github.com/AppleWin/AppleWin/releases/download/v1.31.0.0/AppleWin1.31.0.0.zip)] 

# Navigate to the directory with AppleWin emulator via terminal
``` cd ../YourPath/AppleWin1 ```

# Enter the following command:
``` wine applewin.exe   ```
